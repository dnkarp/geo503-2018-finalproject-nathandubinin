Introduction to problem/question

Proxy data has been used in the creation of paleoclimate models. These include, but are not limited to dendrochronological studies and palynological studies. In the past decade, R has become a powerful tool with statistical analyses, and data visualization to name just a couple ways R has been used. Many different types of research uses R for at least some aspect of research, if it is to plot data simply using ggplot, or to create an online website using markdown or leaflet. It has become the norm within scientiic research transcending disciplines. This is partly because one, it is tremendously powerful as a visualization tool, and two, allows for reproducible results. Reproducibility is arguably one of the most critical aspects in scientific research.

Problem / Question

The purpose of this project is to utilize R to viusalize ancient pollen data in southern Sweden. It will be the groundwork for devloping a climate model specific to the Bohuslän province of Sweden.

Proposed data sources

Data will be acquired through two websites: https://www.pangaea.de/ and http://www.europeanpollendatabase.net/fpd-epd/bibli.do. Both sites offer a plethora of pollen data, but the main focus will be identifying sites within southern Sweden.

Proposed methods

A couple of packages may be employed to make pollen diagrams. The ‘rioja’ package allows for a simplified version of creating different types of pollen diagrams, as opposed to the use of ggplot2. You can use ggplot2 to create pollen diagrams, as is illustrated in the “Example 3” graphic. The ‘rioja’ package however makes it easier. A series of arguements are used, with vectors representing the y and x axis, where the y-axis will represent time or sediment depth (depnding on specific data) and the x-axis representing the percentage of a given species. There are several plot types that are built into ‘rioja’ so, ggplot can still be used, but is not necessry. I will be able to download the pollen data directly from the websites mentioned in the “Proposed data sources” using the ‘rioja’ package.

Expected results

I would like to produce a series of pollen diagrams that represent a series of sites within southern Sweden, with particular attention to the province of Bohuslän. The diagrams will be useful in discussing possible climactic trends through time given different species of vegetation.
